# Linux OS Hybrid Vault: Desktop/Embedded Crunch Engine

The cross-platform computational layer for the Ouroboros framework, supporting desktop development, embedded systems, and simulation.

## Contents

### Python Kernels
- `ouroboros_processor.py` - Core mathematical engine (Möbius, Ramanujan τ, geodesics)
- `zorel_quillan_republic.py` - Living Manifold state machine
- Epistemic discipline framework (20 rules)

### Math Frameworks
- Constraint dynamics: O = E(C(Φ(I))) pipeline
- Von Neumann projection operators
- Banach convergence guarantees
- RK4 numerical integration

### UE5 Desktop Ports
- Non-iOS platform fallbacks
- Simulation mode for thermal/battery testing
- Desktop profiling tools

### Embedded Targets
- Raspberry Pi compatibility layer
- ARM64 optimization paths
- Reduced memory footprint profiles

## Build Requirements
- Python 3.10+
- NumPy, SymPy (see requirements.txt)
- Optional: UE5.3 for desktop builds
