# TOOLBOX README

This directory contains non-essential utilities, scripts, and tools, organized by file format/type (.py, .exe, .sh, etc.). Place helper scripts and program files here which are not required for system boot, but aid in development, testing, or advanced use.

## Folder Structure
- `.py/`   : Python scripts
- `.exe/`  : Compiled binaries/executables
- `.sh/`   : Shell scripts
- `...`    : Add other format-specific folders as needed

Each subfolder should contain only files of its corresponding type.
